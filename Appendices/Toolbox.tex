\section{一些西欧字符的输入}
\par 由于源文件采用的UTF8编码方式，所以一些带修饰的西欧字符可以直接输入，然而这对于长期习惯于英文输入的用户来说，可能会有一些困难。下面列出一些通过命令的方式来产生一些带修饰的字符：
\begin{itemize}
\item \verb"\`a", \`a；
\item \verb"\'e", \'e；
\item \verb"\""o, \"o；
\item \verb"\^u", \^u；
\item \verb"\c{c}", \c{c}；
\item \verb"\v{Z}", \v{Z}。
\end{itemize}
